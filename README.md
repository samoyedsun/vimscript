##### vimscript
- 执行以下脚本:
    ``` shell
    git clone https://github.com/samoyedsun/vimscript.git
    cd vimscript && sh ./install.sh
    ```
- 打开vim输入:PluginInstall然后回撤

- bash提示符
